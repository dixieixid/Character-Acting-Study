We are proud to present the first version of the Agora Community Tools!

This is a selection of production tools developed by Agora.Studio and packaged for the community.

Should be used in Maya 2022 and above.

## Installation: 
To install Agora Community tools, pick one of the following options.

### Drag-and-drop Installer.
1. Drag and drop `agora_community/install.py` file inside a Maya viewport to install or update the tool.

### Maya Module File (`.mod`)
1. Copy `agora_community/agora_community.mod` to your user modules folder:
    - Windows:       `%USERPROFILE%\Documents\maya\modules\agora_community.mod`
    - macOS/Linux:   `~/maya/modules/agora_community.mod`
2. Open the new `agora_community.mod` in a Text Editor.
3. Replace `./agora_community` with the path to the agora_community folder.
4. Restart Maya.

More information and demos: https://agora.community/assets?category=assets&resource=agora-studio-maya-tools
